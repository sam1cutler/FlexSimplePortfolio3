Hello! I'm Sam.

I'm an aspiring software developer based in Seattle, WA. Previously, I have:

  • Worked as an educator, working with college students, high school students, middle school students, and adults, in the United States and China.
  • Done hands-on biology research, designing and executing molecular biology experiments investigating the HIV immune response, stem cell development, lung cancer tumorigenesis, and DNA structure.
  • Used a broad range of technical tools - including Python, Bash, and R - to analyze large bioinformatics data sets, create publication-quality data figures, and communicate scientific results in journal articles and conferences presentations.

Through this diverse set of professional experiences, I have developed great skills in problem-solving, troubleshooting, team collaboration, and communicating complex topics. I will apply all of these skills, along with a rapidly-developing programming ability, to any technical challenges I face as a software developer.

About me

I earned my PhD in molecular and cellular biology from the University of Washington, in the lab of Dr. Toshi Tsukiyama, located at Seattle's Fred Hutch Cancer Research Center. While conducting my PhD research, I used a lot of Python, Bash, and R, and found that I loved programming. In particular, I found it immensely gratifying to use R to create beautiful, publication-quality figures that looked just how I wanted; to write Python scripts that could efficiently analyze huge, whole-genome data sets; and to create Bash scripts that could transform three hours of painstaking, repetitive typing at the command line into a simple process of "run one script, go have lunch, and come back to folders full of neatly analyzed data"! Based largely on that experience, I am now extremely excited to be pursuing a career in software development.

Services I can provide to your team:

  • Creation of HTML pages, incorporating jQuery DOM manipulation, flexbox layouts, responsive navigation, and API requests.
  • Creation of React apps.
  • Help communicating complex, technical information, whether translating customer needs into actionable coding expectations, or translating technical features of code into customer-appropriate user guidelines.
  • Writing Python scripts capable of analyzing biological data, such as genome-wide sequencing results.
  • Writing Bash scripts to allow for radically more efficient running of bioinformatics analytical pipelines.
